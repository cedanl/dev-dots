### Title: Configure Starship Prompt and Bash Dotfiles in the Container

**Description**  
The goal is to integrate the Starship prompt into the containerized environment and set up the necessary dotfiles for both the Starship prompt and Bash. Specifically:
1. Ensure that the Starship prompt is installed and properly configured within the container.
2. Provide dotfiles for:
   - Bash configuration
   - Starship prompt customization  

This will improve the developer experience by offering a cohesive and visually appealing terminal setup out-of-the-box.

**Steps to accomplish:**  
- Include installation steps or add the Starship installation to the container build process.
- Provide or update the `.bashrc` file or equivalent to include relevant Bash settings.
- Provide or update the `starship.toml` file for Starship customization.
